That feel when...

13/12/2014

...you're finally just enough skilled to use something however you want. That's one of the most awesome feelings to me, and really makes me want to keep learning something. Today, I had this realisation with CSS: I just finally got it: I now feel like I can style a page pretty much however I want, and when the CSS doesn't behave as expected, I can fix it myself.

This evening I fixed the projects section, which I had some trouble with yesterday. It turns out, because I had the .column divs within an anchor, they were all the first child of that anchor. The CSS boilerplate I'm using, Skeleton, uses the :first-child pseudoclass to remove the left margin from the first column in a row, but that got applied to all the columns. So I fixed that by putting a div around the anchors, and removing the inner divs, because they weren't really necessary.

I also added some nice transitions to the project buttons, which turned out to be surprisingly easy to do. They now fade to red very nicely when you hover over them. I also changed the body font to Roboto Slab, which looks pretty classy if you ask me. :P

I'm not done with this page yet though, take a look at the to do list above. But I'm a lot less frustrated by CSS now. I still feel a bit meh about web programming though. But that's a topic for another time.
